# Security Policy

## Supported versions

Only the latest release receives fixes.

| Version | Supported |
|---------|-----------|
| v1.x (latest) | ✓ |
| older | ✗ |

## Reporting a vulnerability

**Do not open a public issue.** Use [GitHub Security Advisories](https://github.com/n0troot/SharpGPOwned/security/advisories/new) to report privately. You'll get a response within 7 days.

Include:
- Affected component (GPRecon / GPOwned / Shared)
- Steps to reproduce
- Impact assessment
